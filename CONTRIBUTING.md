# Contributing Guidelines
Follow these when making a pull request to this repository

## Adding new ideas/tasks
**Example** 
- [ ] Task Name [Field]


Write it as
` - [ ] Task Name [Field] `

## Adding/Updating a Completed Project
**Example**
- [x] Task Name[Field]
- Team Member, Month Year


Write it as

`- [x] Task Name[Field]`

`- Team Member, Month Year`
